# 🌟 Overview

**Expense Tracker with Budget Insights** is a modern personal finance management web application that allows users to securely manage their daily income and expenses.

The application provides real-time financial insights using interactive charts, smart budget analysis, PDF & CSV export functionality, and Firebase Authentication.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/Raghav335/Expense-Tracker-with-Budget-Insights.git
```

Move into Project

```bash
cd Expense-Tracker-with-Budget-Insights
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Production Build

```bash
npm run build
```
